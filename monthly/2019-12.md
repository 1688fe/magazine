### 一期一会LeetCode
题目描述：对象扁平化：请实现 flatten(input) 函数，input 为一个 javascript 对象（Object 或者 Array），返回值为扁平化后的结果。

```javascript
var input = {
     a: 1,
     b: [ 1, 2, { c: true }, [ 3 ] ],
     d: { e: 2, f: 3 },
     g: null, 
   }

```

```javascript
输出结果：

{
     "a": 1,
     "b[0]": 1,
     "b[1]": 2,
     "b[2].c": true,
     "b[3][0]": 3,
     "d.e": 2,
     "d.f": 3,
     // "g": null,  值为null或者undefined，丢弃
  }
```


解题详情请看 《[对象扁平化](https://www.yuque.com/1688fe/articles/br77gm) 》
